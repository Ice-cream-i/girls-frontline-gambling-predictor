少前菠菜计算器
=====================================================
数据集
-----------------------------------------------------
使用第五期B服和安卓官服

模型
----------------------------------------------------

* ML.py 里有几种经典的机器学习算法，没做接口，稍改一下也可以用来预测。代码中使用B服训练，安官测试。
* DNN.py 里是一个3层的神经网络， 做了接口（见使用方法）

使用方法
-----------------------------------------------------
pred.py 会读取DNN的预训练模型进行预测。把新的数据添加在feature列表的最后，运行pred.py ，输出的最后一条就是最新的预测。
文件夹中已有预训练模型，不必重新运行DNN.py.
